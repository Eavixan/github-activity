# GitHub Activity CLI

A simple command-line interface application that fetches and displays the recent public activity of a GitHub user in the terminal.

## Project URL

https://github.com/Eavixan/github-activity

## Description

GitHub Activity CLI is a small Node.js project built to practice working with command-line arguments, APIs, JSON data, error handling, Git, and GitHub.

The user provides a GitHub username as an argument, and the app fetches recent public activity from the GitHub API.

Example:

```bash
github-activity kamranahmedse
